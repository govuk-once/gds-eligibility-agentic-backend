## Goal

To create a machine-readable, unambiguous representation of the criteria for eligibility for the UK Skilled Worker Visa.

## Requirements
1. The representation should faithfully and correctly represent the eligibility criteria for the UK Skilled Worker Visa described by the following website, or links from the following website https://www.gov.uk/skilled-worker-visa
2. The representation should be complete enough that it covers all possible edge cases contained within the eligibility criteria.
3. The representation should be unambiguous enough that the outcome is deterministic, and consistent for multiple people with the same personal circumstances
4. The representation should be capable of being visualised in a simple graphical form that it can be easily understood by someone with subject matter expertise of the UK Skilled Worker Visa rules.
